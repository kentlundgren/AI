# AI - Projekt och Exempel

Detta repo innehåller olika AI-relaterade projekt och exempel.

## Innehåll

### 1. Geoffrey Hinton - 10 Bidrag till AI
En interaktiv presentation som visar Geoffrey Hintons 10 viktigaste bidrag till AI-utvecklingen. Presentationen är tvåspråkig (svenska/engelska) och innehåller bilder och förklaringar för varje bidrag.

**Funktioner:**
- Interaktiv bildspel med navigation
- Växla mellan svenska och engelska
- Responsiv design
- Externa bilder från Wikipedia och andra källor

**Teknik:** HTML, CSS, JavaScript
